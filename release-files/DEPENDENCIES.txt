Wine Custom Build - Runtime Dependencies
=========================================

This file lists all runtime dependencies required for Wine to function correctly.

SYSTEM REQUIREMENTS
-------------------
- Ubuntu 22.04 LTS or newer (or Debian-based distribution)
- x86_64 (64-bit) architecture
- 32-bit library support enabled
- ~2 GB free disk space

PREREQUISITE: ENABLE 32-BIT ARCHITECTURE
-----------------------------------------
Before installing Wine, you MUST enable 32-bit architecture support:

    sudo dpkg --add-architecture i386
    sudo apt update


REQUIRED PACKAGES
-----------------
The following packages are required for Wine to run:

Core Libraries:
    libc6:i386
    libc6

X11 and Graphics:
    libx11-6:i386
    libx11-6
    libxext6:i386
    libxext6
    libxrender1:i386
    libxrender1
    libxrandr2:i386
    libxrandr2
    libxi6:i386
    libxi6
    libxcursor1:i386
    libxcursor1
    libxinerama1:i386
    libxinerama1
    libgl1:i386
    libgl1

Fonts and Text:
    libfreetype6:i386
    libfreetype6
    libfontconfig1:i386
    libfontconfig1

Audio:
    libasound2:i386
    libasound2

System Libraries:
    libdbus-1-3:i386
    libdbus-1-3
    libgnutls30:i386
    libgnutls30


INSTALLATION COMMAND
--------------------
Install all required dependencies with a single command:

    sudo apt install -y \
        libc6:i386 \
        libx11-6:i386 libx11-6 \
        libfreetype6:i386 libfreetype6 \
        libfontconfig1:i386 libfontconfig1 \
        libxcursor1:i386 libxcursor1 \
        libxi6:i386 libxi6 \
        libxext6:i386 libxext6 \
        libxrandr2:i386 libxrandr2 \
        libxrender1:i386 libxrender1 \
        libxinerama1:i386 libxinerama1 \
        libgl1:i386 libgl1 \
        libasound2:i386 libasound2 \
        libdbus-1-3:i386 libdbus-1-3 \
        libgnutls30:i386 libgnutls30

-----------------
These packages are recommended for full compatibility:

    libpulse0:i386
    libpulse0

LDAP (For enterprise features):
    libldap-2.5-0:i386
    libldap-2.5-0

Printing:
    libcups2:i386
    libcups2

    ocl-icd-libopencl1:i386
    ocl-icd-libopencl1

Vulkan (Modern graphics):
    libvulkan1:i386
    libvulkan1

Install optional packages:

    sudo apt install -y \
        libpulse0:i386 libpulse0 \
        libldap-2.5-0:i386 libldap-2.5-0 \
        libcups2:i386 libcups2


BUSINESS CENTRAL / BC4 SPECIFIC
--------------------------------
For running Business Central or BC4, you may also need:

    unixodbc
    unixodbc-dev

For SQL Server connectivity:

    curl https://packages.microsoft.com/keys/microsoft.asc | sudo apt-key add -
    curl https://packages.microsoft.com/config/ubuntu/$(lsb_release -rs)/prod.list | \
        sudo tee /etc/apt/sources.list.d/mssql-release.list
    sudo apt update
    sudo ACCEPT_EULA=Y apt install msodbcsql18


VERIFICATION
------------
Check if dependencies are installed:

    # Check multilib support
    dpkg --print-foreign-architectures
    # Should show: i386

    # Check specific package
    dpkg -l libc6:i386
    # Should show: ii libc6:i386 ...

    # Check all required packages
    dpkg -l | grep -E "libc6:i386|libx11-6:i386"


TROUBLESHOOTING
---------------

Error: "Unable to locate package"
Solution: Enable i386 architecture and update:
    sudo dpkg --add-architecture i386
    sudo apt update

Error: "Package has no installation candidate"
Solution: Your Ubuntu version may be too old. Upgrade to 22.04 LTS or newer.

Error: "libGL.so.1: cannot open shared object file"
Solution: Install Mesa OpenGL libraries:
    sudo apt install libgl1:i386 libgl1

Error: "wrong ELF class: ELFCLASS32"
Solution: Install 32-bit C library:
    sudo apt install libc6:i386

DISK SPACE REQUIREMENTS
------------------------
Wine installation: ~2 GB
Wine prefix (per app): ~500 MB - 2 GB
Recommended free space: 5 GB+


CHECKING INSTALLED SIZE
------------------------
After installation, check actual size:

    du -sh /usr/local/lib/wine
    du -sh /usr/local/bin/wine*
    du -sh /usr/local/share/wine
